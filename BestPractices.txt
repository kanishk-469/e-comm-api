REST API Best Practices

>> Using correct status code 
>> Error Handling .. proper error.message
>> Input Validation (express-validator)
>> Using correct request methods(GET,POST,PUT,DELETE)
>> Secure APIs (BasicAuth, JWTAuth etc)
>> Using Naming Conventions for URL 
>> so on...
